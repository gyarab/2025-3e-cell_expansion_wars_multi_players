#Brain Cell Wars

Brain Cell Wars je webová strategická hra pro jednoho nebo více hráčů. Hra se odehrává na obdélníkové herní ploše viděné seshora ve 2D prostředí. Hráč ovládá buňky a postupně zabírá území pomocí chapadel po kterých putují vojáci. Cílem hry je obsadit všechny zbývající buňky na herní ploše a porazit soupeře. Hra nabízí tři singleplayerové úrovně s postupně rostoucí obtížností a multiplayerový režim pro dva a více hráčů. Frontend je napsán v JavaScriptu a backend v Pythonu s frameworkem Django.

## Hra online 

Hra je dostupná na webové adrese https://cellexpansionwars.netlify.app/

## Přidání Hry na Netlify 

1. Registrace na stránce https://netlify.com
2. Najít na hlavní stránce pole:
   > Drag and drop your site folder here

3. Přetáhnout soubory ze souborů do pole -

> Pro představu:
>  index.html
game.html
main.js
login.js
multiplayer.js

4. Netlify vytvoří stránku a vygeneruje adresu ve formátu:
   >https://nahodne-jmeno.netlify.app

5. Konec možnost i přejmenování adresy

## Spuštění ve VS Code 

1. Naklonovat repozitář:
   > git clone https://github.com/gyarab/2025-3e-cell_expansion_wars_multi_players

2. Otevření složky ve VS Code:
> cd 2025-3e-cell_expansion_wars_multi_players
code .

3. Nainstalování rozšíření Live Serve - Kliknutíí na ikonku rozšíření vlevo, vyhledat Live Server a nainstalovat

   4. Otevřít index.html a kliknout pravým tlačítkem a vybrat Open with Live server
   5. Hra se otevře v prohlížeči na adrese http://localhost:5500
