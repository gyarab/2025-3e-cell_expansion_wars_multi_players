### Brain Cell Wars

Brain Cell Wars je webová strategická hra pro jednoho nebo více hráčů. Hra se odehrává na obdélníkové herní ploše viděné seshora ve 2D prostředí. Hráč ovládá buňky a postupně zabírá území pomocí chapadel po kterých putují vojáci. Cílem hry je obsadit všechny zbývající buňky na herní ploše a porazit soupeře. Hra nabízí tři singleplayerové úrovně s postupně rostoucí obtížností a multiplayerový režim pro dva a více hráčů. Frontend je napsán v JavaScriptu a backend v Pythonu s frameworkem Django.

### Brain Cell Wars
Brain Cell Wars is a web-based strategy game for one or more players. 
The game takes place on a rectangular playing field viewed from above 
in a 2D environment. The player controls cells and gradually captures 
territory using tentacles through which soldiers travel. The goal is 
to capture all remaining cells on the playing field and defeat the 
opponent. The game offers three singleplayer levels with gradually 
increasing difficulty and a multiplayer mode for two or more players. 
The frontend is written in JavaScript and the backend in Python using 
the Django framework.

## Play Online
The game is available at https://cellexpansionwars.netlify.app/

## Deploying to Netlify
1. Register at https://netlify.com
2. Find the field on the main page:
   > Drag and drop your site folder here
3. Drag your project files into the field:
   > index.html
   > game.html
   > main.js
   > login.js
   > multiplayer.js
4. Netlify will create the site and generate an address in the format:
   > https://random-name.netlify.app
5. Done — you can also rename the address in site settings

## Running in VS Code
1. Clone the repository:
   > git clone https://github.com/gyarab/2025-3e-cell_expansion_wars_multi_players
2. Open the folder in VS Code:
   > cd 2025-3e-cell_expansion_wars_multi_players
   > code .
3. Install the Live Server extension — click the extensions icon on 
   the left, search for Live Server and install it
4. Open index.html, right-click and select Open with Live Server
5. The game will open in your browser at http://localhost:5500
